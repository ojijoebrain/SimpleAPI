# 📝 SimpleAPI

A basic RESTful API built with **Node.js** and **Express** that supports CRUD (Create, Read, Update, Delete) operations for items.

---

## 🚀 Features

- Get all items
- Get a single item by ID
- Create a new item
- Update an existing item
- Delete an item
